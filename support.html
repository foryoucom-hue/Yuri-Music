❓ Pertanyaan Umum

Bagaimana cara menambahkan YURI MUSIC ke server Discord? +
Klik tombol "Invite Bot" di halaman utama atau gunakan link invite resmi kami. Bot akan meminta izin yang diperlukan, pastikan kamu punya hak "Manage Server". Setelah diinvite, bot langsung siap digunakan dengan slash command /play.
Kenapa bot tidak merespons command saya? +
Pastikan bot memiliki permission yang cukup di channel tersebut, terutama Send Messages, Connect, dan Speak untuk voice channel. Coba juga ketik /help untuk memastikan bot aktif. Jika masih bermasalah, hubungi support kami.
Apakah YURI MUSIC support Spotify? +
Ya! YURI MUSIC mendukung Spotify untuk pengguna Premium. Kamu bisa langsung paste link lagu, album, atau playlist Spotify dan bot akan memutarnya. Pengguna Free dapat menggunakan YouTube dan SoundCloud secara penuh.
Bagaimana cara menggunakan mode 24/7? +
Mode 24/7 tersedia untuk pengguna Premium. Aktifkan dengan mengetik /247 di channel teks manapun. Bot akan tetap berada di voice channel meskipun semua orang sudah keluar, tanpa auto-disconnect.
Bagaimana cara menggunakan audio filter? +
Gunakan command /filter lalu pilih efek yang kamu inginkan dari daftar: bass boost, nightcore, slowed, 8D audio, vaporwave, lo-fi, karaoke, dan lainnya. Filter diterapkan secara real-time tanpa menghentikan lagu yang sedang diputar.
Bagaimana cara membuat dan menyimpan playlist? +
Gunakan /playlist create [nama] untuk membuat playlist baru, lalu tambahkan lagu dengan /play saat playlist aktif. Muat playlist kapan saja dengan /playlist load [nama]. Hapus dengan /playlist delete [nama].
Apakah Premium bisa digunakan untuk beberapa server? +
Premium berlaku per server. Setiap server yang ingin mendapatkan fitur Premium harus berlangganan secara terpisah. Namun untuk plan Lifetime, kamu mendapatkan keuntungan eksklusif tambahan sebagai penghargaan dukunganmu.
Bagaimana cara melaporkan bug atau request fitur? +
Bergabung ke server Discord kami dan gunakan channel #bug-report untuk melaporkan bug atau #feature-request untuk request fitur baru. Sertakan detail lengkap agar tim kami dapat menanganinya lebih cepat.
